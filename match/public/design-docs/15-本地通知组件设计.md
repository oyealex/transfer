# 本地通知组件设计

## 1. 模块职责

本地通知组件位于 common 模块，统一处理邮件、短信和站内信模拟发送。所有业务模块必须通过 `LocalNotificationService` 发送通知。

## 2. 通知类型

| 类型 | 场景 |
|------|------|
| EMAIL | 注册激活、发票通知 |
| SMS | 支付成功、发货提醒 |
| IN_APP | 订单状态、退款状态 |

## 3. NotificationRequest

字段：

| 字段 | 说明 |
|------|------|
| bizType | 业务类型 |
| bizId | 业务 ID |
| receiver | 接收人 |
| channel | 通知渠道 |
| templateCode | 模板编码 |
| variables | 模板变量 |
| idempotencyKey | 幂等键 |

## 4. 发送规则

1. 根据 `idempotencyKey` 去重。
2. 渲染模板。
3. 调用 MockMailSender 或 MockSmsSender。
4. 记录发送日志。
5. 失败时记录失败原因，不影响主业务流程。
