# 非公开黑盒用例使用说明

本目录是组织方使用的非公开 REST 黑盒测试 Maven 项目，不进入选手公开材料。

测试使用 JUnit 5 和 Spring Boot test 支撑，在进程内启动完整 ShopHub 应用，并且只通过 HTTP API 与被测系统交互。

每个测试方法都会获得新的 Spring Boot 测试上下文和随机 H2 内存库。业务前置条件仍通过 REST API 创建，但用例隔离不依赖应用中的 reset/bootstrap 端点。

## 运行

从仓库根目录执行。先安装基础工程：

```bash
mvn -f public/code/pom.xml install
```

运行全部非公开黑盒用例：

```bash
mvn -f private/test-cases-internal/pom.xml test
```

运行单个非公开测试类：

```bash
mvn -f private/test-cases-internal/pom.xml -Dtest=HiddenL2Test test
```

本模块有意不加入 `public/code/pom.xml` 的 `<modules>`。运行这些用例不需要修改业务工程 POM。
